# 🚀 Nexium — Digital Twin for Emergency Evacuation Planning

## 🧠 Problem Statement
Crowd mismanagement during emergencies leads to congestion, delays, and risk to human life. Existing systems lack intuitive simulation tools for planning evacuation strategies.

## 💡 Solution
We developed an immersive 3D digital twin that simulates crowd movement during emergencies and optimizes evacuation paths using adaptive routing.

## 🎯 Features
- Real-time crowd simulation
- Congestion visualization (color-based)
- Emergency trigger scenario
- AI-based evacuation optimization
- Top view + 3D perspective toggle

## 🛠️ Tech Stack
- Unity (3D Simulation)
- C# (Agent Logic)
- XR Toolkit (optional)
